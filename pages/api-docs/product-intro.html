光宇云产品介绍

一、产品定位

光宇云存储平台是一套面向企业、开发者及各类数据密集型业务的分布式云存储与数据服务体系。

平台基于分布式架构与弹性资源调度，支持大规模横向扩展、多副本冗余存储、跨地域自动容灾，能够满足从对象存储、文件存储的多样化需求。

与传统中心化云存储相比，光宇云通过节点自治与协作机制，彻底消除了单点瓶颈与故障风险，实现 更高的数据可靠性、更灵活的弹性扩展能力以及更低的总体拥有成本（TCO）。

二、架构设计

  1. 去中心化架构

    节点对等互联：所有存储节点均以对等角色参与数据存储与服务提供，无需依赖中心节点，避免单点故障。

    弹性扩展：节点可随时动态加入或退出集群，系统自动感知并重构数据分布，以支撑业务量的快速增长与缩减。

    跨地域分布：数据副本分散存储在不同物理机房或区域级节点，保证地理级别的高可用性。

  2. 多副本冗余与一致性机制

    双副本标准：默认每份数据至少存储两个副本，确保在硬件或网络异常情况下仍可快速恢复。

    一致性协议：支持 Raft、Paxos 等强一致性协议，也可选择最终一致性策略，兼顾性能与可靠性。

    智能副本调度：根据节点负载、网络带宽和物理位置动态调度副本，提高整体访问效率。

  3. 数据访问与安全

    统一接口：提供 RESTful API 与多语言 SDK，支持与现有业务系统无缝集成。

    多层安全防护：内置 ACL 权限管理、SSL/TLS 加密传输机制，以及数据静态加密存储。

    安全防御能力：内置防御 DDoS、暴力破解及恶意访问的安全策略，保障服务连续性。

    三、核心能力

    对象存储（Object Storage）：支持 PB 级非结构化数据存储，适用于文件、图片、视频、日志等场景。

    文件存储（File Storage）：支持企业应用、共享文档和开发环境的文件级访问需求。

    CDN 加速：深度集成内容分发网络，全球范围内智能调度，保障低延迟访问体验。

    跨协议兼容：支持 HTTP/HTTPS、S3、FTP 等多种访问方式，兼容现有生态系统。

    应用支撑：为上层 SaaS、PaaS 平台提供高可用、可扩展的底层存储资源池。

    四、应用场景

    企业级备份与容灾：多副本跨地域存储，分钟级数据恢复能力，降低业务中断风险。

    视频点播与直播：结合 CDN 加速，实现高并发访问与低延迟传输，适合媒体与娱乐行业。

    大数据与 AI 分析：支持 Hadoop、Spark 等计算框架直接访问存储，计算与存储解耦，提升分析效率。

    互联网与移动应用：支撑亿级用户高并发访问，提供弹性扩容能力应对业务高峰。

    政企与金融场景：满足合规性要求，提供高安全级别的数据存储与访问控制。

    五、优势

    高可靠：多副本 + 智能容灾机制，数据持久性高达 99.9999%。

    高性能：支持 PB 级数据存储，轻松应对百万级并发请求。

    高安全：全链路加密、分级权限管理与入侵检测，构建全方位安全体系。

    低成本：按需扩展、资源弹性调度，大幅降低企业硬件与运维成本。

    开放兼容：支持主流开发框架与协议，企业可平滑迁移与快速集成。
