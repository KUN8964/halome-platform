大文件上传（支持断点续传）

简要描述
  • 大文件上传（支持断点续传）
请求URL
  • https://gw-hangzhou-2.halome.cc/v1/addLargeFile
请求方式
  • POST
Head 参数
参数名 必选 类型 说明
FileStartIndex 是 string 文件切片当前的位置（单位字节） 切片大小可选范围 1M-10M）
FileSize 是 string 文件大小 单位字节
FileName 是 string 文件名称
FileMd5 是 string 文件MD5值
AuthToken 是 string token
Body 参数
参数名 必选 类型 说明
是 application/octet-stream 文件二进制流
返回示例 （同步）
{
    "code":200,
    "message":"",
    "fileIndex":0,
    "id":""
}
返回参数说明
参数名 类型 说明
code int 200 成功
message string 消息内容
fileIndex int64 当前上传文件位置索引（单位字节）
id string 文件ID(最后上传完成才会有)
错误码

200 成功

1 FileStartIndex 参数错误

2 FileSize 参数错误

3 token error

4 参数错误

5 创建文件失败

6 写入文件失败

7 文件索引错误

8 拷贝数据失败

9 文件MD5错误

10 上传失败

11 上传到分布式存储系统失败

12 上传数据库失败
