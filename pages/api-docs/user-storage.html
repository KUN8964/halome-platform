获取用户空间和流量相关信息

简要描述
  • 获取用户空间和流量相关信息
请求URL
  • https://gw-hangzhou-2.halome.cc/u/getUserStorageInfo
请求方式
  • GET
Head 参数
参数名 必选 类型 说明
AuthToken 是 string token
返回结果
{
    "code":200,
    "msg":"success",
    "data":{
        "Account":"",
        "SpaceName":"",
        "SpaceSize":11112123,
        "SpaceUsed":123123,
        "SpaceExpirationDate":123123213,
        "FlowSize":213123312,
        "FlowUsed":12312312,
        "FlowExpirationDate":170123123,
        "Status":0
    }
}
参数名 类型 说明
Account string 账号
SpaceName string 空间名称
SpaceSize int 购买空间大小（单位字节）
SpaceUsed int 空间使用量（单位字节）
SpaceExpirationDate int 空间到期时间（时间戳）
FlowSize int 购买流量大小（单位字节）
FlowUsed int 流量使用量（单位字节）
FlowExpirationDate int 流量到期时间（时间戳）
Status int 账号状态（0正常，1异常）
备注
  • 无
