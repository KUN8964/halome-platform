获取token

简要描述
  • 获取token
请求URL
  • https://gw-hangzhou-2.halome.cc/u/createToken
请求方式
  • POST
参数
参数名 必选 类型 说明
account 是 string 账号
apiKey 是 string apiKey
expireTime 是 integer 到期时间戳 
{
    "account": "13799988777",
    "apiKey": "2313sadasdasdasda123121ewwe",
    "expireTime": 1987888888
}
返回示例
 {
    "code":"200",
    "message":"success",
    "data":"token"
}
返回参数说明
参数名 类型 说明
code int 200 成功
message string 消息内容
data string token
备注
  • 无
