API使用流程

  1. 账号注册与认证

    用户需在光宇云平台注册企业账号，并提交相关资质资料完成企业实名认证。（当前阶段暂不支持个人实名认证。）

  2. 创建云空间并获取密钥

    完成认证后，可在管理控制台中创建专属云空间。系统会为用户生成并展示对应的 API Key，作为访问凭证的基础。

  3. 申请与获取token

    使用 API Key 通过认证接口申请临时访问 Token。该 Token 具有时效性与安全属性，用于后续所有 API 请求。

  4. 接入API与调用

    在业务系统中集成光宇云提供的 RESTful API 或多语言 SDK，并在请求中携带有效的 Token，即可完成与平台存储及数据服务的对接。
